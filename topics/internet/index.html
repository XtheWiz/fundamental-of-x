Fundamental of Internet

The plumbing under every URL. We start at "user types a URL" and walk down the stack: DNS resolves the name, IP routes the packets, TCP keeps them in order, TLS encrypts them, HTTP carries the meaning. Each lesson is an interactive widget you can poke.

This topic owns the transport story. The HTTP semantics (verbs, status codes, REST) live in Fundamental of Backend when that ships.
