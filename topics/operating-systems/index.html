Fundamental of Operating Systems

Every program you write runs on top of a kernel that's juggling dozens of others. The OS arbitrates CPUs, memory, files, devices, and processes so they all coexist without stepping on each other. It's the layer your code answers to and rarely sees.

Pairs naturally with the Computer Architecture topic — that one's the hardware view, this one's the kernel view.
