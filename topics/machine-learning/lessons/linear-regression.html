Linear Regression

The simplest model worth its name. Two numbers — a slope and an intercept — fully define the function. Adjust them by hand and watch the loss respond. This is the foundation everything else builds on.

The model

ŷ = w · x + b. Two parameters: weight w, bias b. For each data point (x, y), the prediction error is ŷ − y. The loss we minimize is usually the mean squared error: L = mean((ŷ − y)²).

Squaring keeps errors positive and penalizes big misses more than small ones. Mean averages across all points.

The optimization

Linear regression is one of the rare cases where the optimal (w, b) can be computed analytically — set derivatives to zero, solve. But that's a luxury; almost every other model needs iterative optimization. Get a feel for the loss surface here, and gradient descent (next lesson) makes intuitive sense.

Try it

Move the weight and bias sliders. Watch the line and the loss change together.

Key takeaways

  • Linear regression's loss surface is a perfect bowl (a paraboloid) — one minimum, easy to find.
  • Two parameters → two-dimensional loss surface. Easy to visualize. Most real models have millions; the principle is the same.
  • "Linear" doesn't mean "boring." Add log/exp transforms to x and you've got a surprising amount of modeling power without leaving the linear family.
  • Used everywhere as a baseline: every ML pipeline should start with linear regression to see what naïve fitting gets you. Often more than expected.
