Neural Networks

Stack linear models, put a non-linear squash between them, and you get a function that can approximate almost anything. That's the entire neural-network idea. The depth and width of modern models is just more of the same.

A neuron

One neuron computes output = σ(w₁x₁ + w₂x₂ + … + b) — a weighted sum of inputs passed through an activation function (sigmoid, tanh, ReLU). That's logistic regression in one neuron's worth of code.

A layer, a network

A layer is several neurons in parallel, each with its own weights, all looking at the same inputs. A network is layers stacked: the outputs of one layer feed the next.

With one hidden layer and enough neurons, a network can approximate any continuous function (the universal approximation theorem). In practice we use many shallow-ish layers because they train better.

Why non-linearity?

Without the activation function, stacking layers collapses to one layer: linear-of-linear is still linear. The squash makes each layer add expressive power. The whole reason XOR was "unsolvable" in the 1960s AI winter was that researchers used single-layer perceptrons — no hidden layer, no curve.

Key takeaways

  • "Neural network" is misleading marketing — they're not very brain-like. They're stacked weighted sums with squashes.
  • Each layer transforms its input into a representation the next layer can use. Deep networks build hierarchies — edges → shapes → objects.
  • The biggest practical change since the 1990s: ReLU replaced sigmoid/tanh (better gradients), GPUs replaced CPUs (1000× speed), and we trained on internet-scale data.
  • Modern architectures (CNNs, transformers, diffusion) reuse this same backbone with structured connectivity. Same math, smarter wiring.
